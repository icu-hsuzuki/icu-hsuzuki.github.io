DONKEY

(Installation using Terminal)
  1. Download donkey-current.tar.gz from: ftp://ftp.aist-nara.ac.jp/pub/Security/tool/donkey/
  2. % tar zxf donkey-current.tar.gz
    (Stuff-it expander may have expanded the file on your desktop already.)
  3. % cd donkey-0.5
  4. % ./setup Darwin
  5. % cd obj/Darwin/
  6. % ./configure
  7. % make
  8. % sudo make install
  9. % Password: XXXXXXXX
  10. % rehash
  11. % donkey -f md5 nnn xxnnnn
