Macintosh

  • SSH for Macintosh : MRJ SDK + Mindterm + BetterTelnet/Fetch (2001.7.25)
    湘南藤沢インフォメーションテクノロジーセンターの「MacintoshでのSSHを使った接続方法」 でほとんど尽きていますが、このままだと苦労した部分がありますので、それを書き換えておきます。
    1. MRJ2.2.5 および MRJ Software Development Kit (SDK) 2.2 を Download してこの順番で Install :
      Download Site http://developer.apple.com/java/download.html
    2. MindTerm2.0 (MindTerm application binaries) を Download :
      Download Site http://www.appgate.org/products/mindterm/personal/mindterm_downloads.html
    3. BetterTelnetJ 2.0 を Download して Install :
      Download Site http://www.vector.co.jp/soft/mac/net/se061545.html
    4. MRJ SDK 2.2:Tools:Application Builders:JBindery を起動
    5. Open で mindterm_2.0-bin:mindterm.jar を開く
    6. Class nameを、com.mindbright.application.MindTerm に変更
    7. Save as で、Save as application にチェックを入れたまま Mindterm2 などと名前をつけて、好きな場所に保存。後のことを考えると BetterTelnetJ Folder 内などもおすすめ。

    MindTermの使い方(基本)：

    1. Mindterm2 を起動
    2. File menu から Connect を選び New Server ボタンをおして表示されたウィンドウに、Server [Username, Password, Save as alias ボタン] 入力、Connect ボタンをおして Connect
    3. Tunnelsメニュー→Basicを選ぶ
    4. Local port: 8888、Protocol: telnet、Remote host: 接続先ホスト名、 Remote port: 23と入力し[telnet を選ぶと自動的に入る]、Addボタンを押す
    5. BetterTelnetを起動
    6. Fileメニュー→Open connectionを選び、接続先をlocalhost:8888と 入力
    7. 指定した接続先ホストのログインプロンプトが表示されるので、ここでログイン名、パスワードを入力すれば、ログインして BetterTelnetJ の機能[日本語など]を telnet したときと同じように扱うことができる
    ２回目からは、２で Save as alias ボタンを選んであれば、alias をいれ、すぐ接続できる。

    MindTermを経由のftp

    1. Mindterm2.appを起動
    2. remote host に接続
    3. Tunnelsメニュー→Basicを選ぶ
    4. Local port: 8889 (任意ですが、うまく行かないときは別の数にしてみてく ださい)、Protocol: ftp、Remote host: 接続先ホスト名、 Remote port: 21と入力し、Addボタンを押します。
    5. Fetchを起動し、Customizeメニュー→Preferences 設定ダイアログを開く
    6. FirewallタブでUse passive mode transfers(PASV)にチェックを入れる
    7. Fileメニュー→New connectionを選び、接続先をlocalhost:8889と 入力し、ログイン名、パスワードを入力する
