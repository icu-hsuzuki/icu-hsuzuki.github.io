Additional Information

  • The present version should work with the old style files of AMS-LaTeX. If you have any inconvenience or found some bug, please write to Hiroshi Suzuki (hsuzuki@icu.ac.jp).
  • Dr. Akihiro Munemasa reported us that the style file aspmproc.sty with some old version of AMS-LaTeX files (amsproc.cls, amsthm.sty) produces error. He suggests to use the newest AMS-LaTeX files at AMS TeX Resources Home Page.
    • The latest version of amsproc.cls : version = "1.2r", date = "1997/03/26"
    • The latest version of amsthm.sty : version = "1.2d", date = "1996/10/29"

For any question, please feel free to write to Hiroshi Suzuki (hsuzuki@icu.ac.jp).
