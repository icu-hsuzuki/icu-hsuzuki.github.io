TeX

  • 「もっと AMS-TeX」野寺孝志著、共立出版株式会社 (ISBN4-320-02538-5, 1991.2.5)
    個人的には AMS-TeX を使って論文を書いたことはないが、アメリカの研究者は今も、AMS-TeX を使う人が多いため、論文のやり取り、編集、共同研究等のためには、利用することもある。日本語で書かれた数少ない参考書。

  • 「LaTeX 入門 -- 美文書作成のポイント」奥村晴彦監修、技術評論社 (ISBN4-7741-0091-9, 1994.12.1)
    出版当時の、TeX, LaTeX に関連する情報が詰まっている。自分用のマクロの作り方が詳しく説明されているので、利用することが多い。

  • 「日本語 LaTeX2eブック」中野賢著、アスキー出版局 (ISBN4-7561-1667-1, 1996.11.1)
    アスキー版 pLaTeX 2.1.4 が出たときに合わせて出版された解説書。pLaTeX になって、非常に機能がました。この書もその解説書として、良く書かれている。しかし、英語版との互換性はいろいろと問題をはらむようになって来ている。研究者は注意して利用すべきである。

  • 「LaTeX2e階悌」藤田眞作著、アジソン・ウェスレイ・パブリッシャーズ・ジャパン株式会社 (ISBN4-7952-9685-5, 1996.11.20)
