\begin{enumerate}
\item $L = \bQ(\sqrt{3},i),\; (i = \sqrt{-1})$ とする。このとき以下の問いに答えよ。
     \begin{enumerate}
     \item 拡大 $L:\bQ$ は、単純拡大であることを示せ。すなわち、$L$ のある元 $\alpha$ で、$L = \bQ(\alpha)$ となるものを見つけよ。
     \item 上で求めた $\alpha$ について、その $\bQ$ 上の、最小多項式を求めよ。
     \item $L$ の自己同型写像で、$\sqrt{3}$ を、$\sqrt{3}i$ に移すものは存在するか判定し、その理由も述べよ。（$L$ の自己同型写像は、$\bQ$ の元を、動かさないことは、証明せずに使っても良い。）
     \item 体 $L$ は、上で求めた $\alpha$ の最小多項式の $\bQ$ 上の分解体であることを示せ。
     \item 下の定理を用いて、$L$ の自己同型写像 $\tau$ で、$\tau(\sqrt{3}) = -\sqrt{3}$、$\tau(i) = -i$ を満たすものが存在することを示せ。
     
     \vspace{2ex}
     \noindent
     {\bf 定理}  $\sigma : K \longrightarrow L$ を、体の同型とし、$K(\alpha):K$ 及び $L(\beta):L$ を、体の拡大で、$\alpha$ は、$K$ 上代数的、$\beta$ は、$L$ 上代数的とする。$\alpha$ の $K$ 上の最小多項式を $m(t)$ としたとき、もし、$\beta$ の $L$ 上の最小多項式が、$\hat{\sigma}(m(t))$ （$m(t)$ の係数を、$\sigma$ で移して出来た $L$ 係数の多項式）ならば、$K(\alpha)$ から $L(\beta)$ への、体同型 $\tau$ で、$K$ 上では、$\sigma$ でかつ、$\tau(\alpha) = \beta$ となるものが存在する。

     \end{enumerate}
     
\item $F$ を 81個の元からなる体とする。このとき以下の問いに答えよ。
     \begin{enumerate}
     \item $\phi$ を、次のような写像とする。
     $$\phi:\bZ \longrightarrow F \; (n \longmapsto n\cdot 1)$$
     このとき、$K = Im(\phi) = \phi(\bZ)$ は、$3$ 個の元からなる体となることを示せ。
     \item $F$ の元は全て、方程式 $t^{81}-t = 0$ の根であることを示せ。
     \item $\sigma$ を、次のような写像とする。
     $$\sigma: F \longrightarrow F \; (x \longmapsto x^3)$$
     このとき、$\sigma$ は、体 $F$ の自己同型写像であることを示せ。
     \item 拡大 $F:K$ は、ガロア拡大（正規かつ分離拡大）であることを示せ。
     \item 拡大 $F:K$ のガロア群は、$\sigma$ で生成される位数 $4$ の巡回群であることを示せ。
     \end{enumerate}

\item $\alpha = ^4\!\!\!\!\sqrt{3}$ (すなわち、$\alpha$ は、$4$ 乗すると $3$ になる、正の実数）とし、$K = \bQ(\alpha)$ とする。このとき、以下の問いに答えよ。
     \begin{enumerate}
     \item $K$ の $\bQ$ 上の拡大次数 $[K:\bQ]$ を求めよ。
     \item $K$ は、$\bQ$ 上正規拡大ではないことを示せ。
     \item $K$ の $\bQ$ 上の正規閉包で $\bC$ に含まれるものを $L$ とする。このとき、$L$ の $\bQ$ 上のガロア群 $\Gamma(L:\bQ)$ の位数を求めよ。
     \item 拡大 $L:\bQ$ の中間体 $M$ で、$[M:\bQ] = 2$ となるものはいくつあるか。
     \item 上の条件を満たす中間体を全て求めよ。
     \end{enumerate}
