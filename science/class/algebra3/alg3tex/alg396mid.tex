解答は、レポート用紙または、ルーズリーフノートに書き、ID 番号、氏名、問題番号も忘れずに書いて下さい。提出期限は、１月２１日午後１時半までとします。($6\times 14 + 16 = 100$)

\begin{enumerate}
\item $L = \bQ(\sqrt{3},i),\; (i = \sqrt{-1})$ とする。このとき以下の問いに答えよ。
     \begin{enumerate}
     \item 拡大 $L:\bQ$ は、単純拡大であることを示せ。すなわち、$L$ のある元 $\alpha$ で、$L = \bQ(\alpha)$ となるものを見つけよ。
     \item 上で求めた $\alpha$ について、その $\bQ$ 上の、最小多項式を求めよ。
     \item $L$ の自己同型写像で、$\sqrt{3}$ を、$\sqrt{3}i$ に移すものは存在するか判定し、その理由も述べよ。（$L$ の自己同型写像は、$\bQ$ の元を、動かさないことは、証明せずに使っても良い。）
     \item 体 $L$ は、上で求めた $\alpha$ の最小多項式の $\bQ$ 上の分解体であることを示せ。
     \item $L$ の自己同型写像 $\tau$ で、$\tau(\sqrt{3}) = -\sqrt{3}$、$\tau(i) = -i$ を満たすものが存在することを示せ。
     \end{enumerate}
     
\item $K$ を ２個の元からなる体（$\simeq\bZ_2$）とし、$m(t) = t^4+t+1\in K[t]$ とする。このとき以下の問いに答えよ。
     \begin{enumerate}
     \item $m(t)$ は、$K$ 上の、既約多項式であることを示せ。
     \item $L = K[t]/(m(t))$ は、１６個の元からなる体であることを示せ。
     \item $L$ を、$K$ の拡大体と見たとき、この拡大は、正規拡大であることを示せ。
     \item $t^3+t+1$ は、$L$ 内に根を持たないことを示せ。
     \end{enumerate}

\item $\alpha = ^4\!\!\!\!\sqrt{3}$ (すなわち、$\alpha$ は、$4$ 乗すると $3$ になる、正の実数）とし、$L = \bQ(\alpha)$ とする。
	\begin{enumerate}
	\item $(\alpha+1)^{-1}$ を $\alpha$ の３次以下の多項式で表せ。
	\item $L$ の $\bQ$-自己同型をすべて求めよ。
	\item $L$ は、$\bQ$ 上正規拡大ではないことを示せ。
	\item $L$ の拡大体で $\bC$ に含まれ、$L:\bQ$ が正規拡大となるものの内、拡大次数が、最小となるものは、ただ、一つしかないことを示せ。
	\end{enumerate}

\item 
	\begin{enumerate}
	\item 「代数学の基本定理 」（複素数体 $\bC$ 上の多項式は、$\bC$ 上で、分解する）を証明せよ。(16pts)
	\item $L$ を $\bC$ の代数拡大とすると、$L = \bC$ であることを示せ。
	\end{enumerate}
 \end{enumerate}
